📖 About the Program
This application is a full-stack student management system developed using the MVC architecture.

Model: Handles database schema using MongoDB
View: Uses EJS templates for UI rendering
Controller: Contains business logic and request handling
The system supports complete CRUD operations:

Add Student
View Students
Edit Student
Delete Student
This architecture ensures proper separation of concerns and scalable code structure.

⚙️ Features
Add new students
Edit existing records
Delete students
Dynamic rendering using EJS
Clean MVC structure
Dark mode UI
🧠 MVC Workflow
Client → Routes → Controller → Model → Database  
       → Controller → View (EJS) → Response
