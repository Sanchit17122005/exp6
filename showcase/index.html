Student Management System

A full-stack MVC application built using Node.js, Express, MongoDB, and EJS. Demonstrates CRUD operations with proper architecture separation.

🚀 Features

  • Add Student
  • Edit Student
  • Delete Student
  • MVC Architecture
  • MongoDB Integration

⚙️ Tech Stack

Node.js • Express.js • MongoDB • EJS • MVC Pattern
